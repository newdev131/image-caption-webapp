# AI Image Caption Web App

## Live Demo
https://ai-caption-generator-byarfa.netlify.app

## What it does
Upload any image and get AI-generated captions.

## How to Use
1. Upload an image (click or drag)
2. Choose number of captions (1-5)
3. Choose style (Creative, Poetic, Cinematic, Funny)
4. Click "Generate Captions"
5. Copy your favorite caption

## Features
- Upload images
- 1-5 captions per image
- 4 caption styles
- Copy to clipboard

## Technologies
- HTML
- CSS
- JavaScript

## API Used
This app connects to: https://arfanaeem-image-caption-api.hf.spac
